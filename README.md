# KK Music Festival Archive Site

Open `index.html` in a browser to view the archive.

The site is plain HTML, CSS, and JavaScript:

- `index.html` is the page structure.
- `styles.css` controls the visual design.
- `script.js` renders search, filters, cards, image highlights, and folder summaries.
- `data/files.js` is the generated list of files from the zip.
- `files/` contains the extracted real files that the cards open.

Hidden macOS helper files from the zip, such as `__MACOSX` and `._` files, were not included.
